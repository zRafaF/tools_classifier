# Convenção de nomenclatura dos datasets

O nome dos datasets indicam os seus processamentos, escolha um que se adéque ao seu uso.

- `c`: Cropped (recortado)
- `r`: Resized (redimensionado)
- `a`: Augmented (aumentado)
- `u`: Unsorted (não classificado)
- `g`: Grouped (agrupado)
- `png`: Formato PNG
- `jpg`: Formato JPG
